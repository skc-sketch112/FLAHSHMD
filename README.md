## ✨ Features
- ✅ Plugin System (add `.js` files inside `plugins/`)
- ⚡ Fast & lightweight WhatsApp bot
- 📝 Customizable stylish **menu**
- 🖼️ Image search (`!img`)
- 🎵 Song downloader (`!song`)
- 🛠️ Easy to extend with new commands
